\section{Introduction} \label{sec:intro}
%------------------------------------

\begin{itemize}
    \item Recent work from \citet{keatingCrosslanguageAcousticSpace2023} has shown that voice quality across languages map very similarly.
    \item This handout takes the work from \citet{keatingCrosslanguageAcousticSpace2023} and applies it to Santiago Laxopa Zapotec.
    \item The goal is to see how well SLZ compares to the results of \citet{keatingCrosslanguageAcousticSpace2023} and how residual H1 functions in these analyses.
    \item This analysis uses the same NMDS analysis as \citet{keatingCrosslanguageAcousticSpace2023} and the a more robust method of doing classification and regression tress known as bootstrap aggregating or bagging \citep{breimanBaggingPredictors1996,hastieElementsStatisticalLearning2009}.
\end{itemize}

%------------------------------------
\section{Preparing the data for analysis} \label{sec:data}
%------------------------------------

\begin{itemize}
    \item The data came from fieldwork conducted in Santiago Laxopa, Oaxaca, Mexico in 2022.
    \item The data was identical to the data used in \citet{brinkerhoffResidualH1Measure2024}.
    \item The data was preprocessed in the same way as \citet{keatingCrosslanguageAcousticSpace2023}.
    \begin{itemize}
        \item This consisted of extracting the middle portion of each vowel from the data. 
        \item This was done to ensure that the data was as close to the steady state portion of the vowel as possible. 
        \item After extracting the middle portion of the vowel, the associated acoustic measures were z-scored to bring everything into the same frame of reference. 
        \item Because the data was z-scored, some values were negative which produces a problem for the NMDS analysis.
        \item To counteract this, the lowest value was added to all the data to make all the values positive while still maintaining the same relative distance between the values.
    \end{itemize}
\end{itemize}
